# Ek Gelir Akademi — Pasif Gelir Yaratma Premium Sunum

Tek dosyalık, kurulum gerektirmeyen statik bir web sunumudur.

## Nasıl Açılır?

**1. Yöntem — Çift tıkla:**
- `index.html` dosyasını bilgisayarınızda çift tıklayın. Tarayıcıda doğrudan açılır. Bu kadar.

**2. Yöntem — Sürükle-bırak:**
- `index.html` dosyasını Chrome, Edge, Safari veya Firefox penceresine sürükleyin.

**3. Yöntem — Web sunucusunda yayınla:**
- Dosyayı herhangi bir statik hosting servisine yükleyin (GitHub Pages, Netlify, Vercel, Cloudflare Pages, kendi sunucunuz...). Klasörde sadece `index.html` ve bu `README.md` yeterlidir.

## İçindekiler
- Sol menüden 13 ekrana (1 Açılış + 12 bölüm) tıklayarak gezinebilirsiniz.
- Alt kısımdaki ileri/geri butonları ile sıralı ilerleyebilirsiniz.
- Klavyede **← →** okları da bölüm gezintisi yapar.
- Mobilde sol üstteki menü düğmesinden bölümlere erişin.

## Bağımlılıklar
- **Hiçbir build aracı yok.** Node.js, npm, yarn gerekmez.
- Yazı tipleri (Fraunces + Manrope + JetBrains Mono) Google Fonts üzerinden internetten yüklenir.
- Çevrimdışı kullanmak isterseniz fontları yerel olarak indirip `<link>` etiketini güncelleyebilirsiniz.

## Lisans / Not
Bu sunum bilgilendirme amaçlıdır; finansal tavsiye niteliği taşımaz.
